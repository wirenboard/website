# Добавление обучающих видео

Содержимое папки `content/[locale]/video-education` автоматически формирует список обучающих видео на сайте:
* [content/ru/video-education](/content/ru/video-education) — [Обучающие видео](https://wirenboard.com/ru/contents/video-education/) в русской версии;
* [content/en/video-education](/content/en/video-education) — [Educational videos](https://wirenboard.com/en/contents/video-education/) в английской версии.


### 1. Подготовка файлов

Перейдите в папку нужного языка:
* Английский [content/en/video-education](/content/en/video-education)
* Русский [content/ru/video-education](/content/ru/video-education)

Создайте структуру файлов:
* `content/[lang]/video-education/название_ролика.md` — файл с метаданными видео
* `content/[lang]/video-education/название_ролика/` — папка для обложки

### 2. Формат файла видео

Создайте `.md` файл со следующим содержимым:

```yaml
---
title: Название обучающего видео
url: https://www.youtube.com/embed/VIDEO_ID
cover: название_ролика/обложка.webp
date: 2024-12-22
chapters:
  - time: "00:00"
    title: "Введение"
  - time: "01:30"
    title: "Основная часть"
  - time: "03:45"
    title: "Заключение"
---
```

### 3. Параметры видео

| Параметр | Обязательный | Описание |
|----------|--------------|----------|
| `title`  | Да | Заголовок видео |
| `url`    | Да | Ссылка на YouTube видео в формате embed |
| `cover`  | Да | Путь к обложке видео (размещать в подпапке с тем же именем) |
| `date`   | Да | Дата публикации в формате YYYY-MM-DD |
| `category` | Нет | Категория видео (по умолчанию: `educational`) |
| `chapters` | Нет | Массив глав с таймкодами |

### 4. Требования к файлам

**Обложка:**
- Размещать в подпапке с тем же именем, что и `.md` файл
- Формат: WebP (рекомендуется), JPG или PNG
- Рекомендуемый размер: 1280×720 пикселей
- Имя файла: `обложка.webp` (или другое с указанием правильного расширения)

**Именование файлов:**
- Использовать только латиницу
- Заменять пробелы на подчёркивания `_`
- Не использовать специальные символы (тире, скобки, решётки и т.д.)

## Предпросмотр

Для предпросмотра изменений в VSCode можно использовать плагин, либо запустить сайт локально:
- http://localhost:3000/ru/contents/video-education (русская версия)
- http://localhost:3000/en/contents/video-education (английская версия)

## Публикация

Изменения станут доступны после отправки пул-реквеста и его утверждения.

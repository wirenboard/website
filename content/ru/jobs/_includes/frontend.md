::vacancy

#title
Middle/Senior Frontend разработчик

#salary
200 000 — 250 000 ₽/мес

#description
Мы ищем талантливого frontend разработчика для создания интуитивно понятных и функциональных веб-интерфейсов для управления нашими устройствами. Кандидат будет тесно сотрудничать с командой backend-разработчиков и продакт оунерами для обеспечения наилучшего пользовательского опыта и интеграции с облачными сервисами.

#goals
* развивать web-интерфейс для управления контроллерами (сейчас написан на AngularJS);
* в перспективе создание web-сервисов с нуля (опыт backend или full-stack разработки приветствуется).

#requirements
* опыт разработки web-приложений (HTML, CSS, JS);
* знание современных фреймворков;
* умение читать чужой код (даже ужасный);

#advantage
* знание AngularJS.

#conditions
* удалённая работа либо работа в офисе в Долгопрудном (5 км от МКАД) с кофемашиной и печеньками;
* свободный график.

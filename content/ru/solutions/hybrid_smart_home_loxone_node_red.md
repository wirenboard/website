## Технические решения

### Архитектура системы

Основной контроллер гибридной системы — Loxone Miniserver. Часть оборудования подключена к нему напрямую, а часть — через контроллер Wiren Board. Взаимодействие между контроллерами реализовано по Modbus TCP.

Не все оборудование поддерживается в Loxone. Для следующих задач не было готовых решений:

- управление Apple TV голосовыми командами через Алису;  
- интеграция Wi-Fi-штор через платформу Tuya SmartLife;  
- перекодирование видеопотоков от камер для отображения в интерфейсе Loxone.

Чтобы решить эти задачи, на Wiren Board установили среду визуального программирования Node-RED. В ее репозитории есть готовые ноды (пакеты) для работы как с конкретными устройствами, так и с различными протоколами и сервисами.

Например, интерфейс Loxone поддерживает видеопоток в формате MJPEG, тогда как камеры передают поток по RTSP (обычно в кодеке H.264/H.265). Чтобы обеспечить совместимость, в Node-RED реализовали сценарий преобразования видеопотока в MJPEG. При этом конвертация запускается только в момент просмотра камеры, чтобы не создавать лишнюю нагрузку на контроллер.

### Микроклимат

Основная система отопления — водяной теплый пол. Для управления теплоотдачей на коллекторы установлены сервоприводы, подключенные к контроллеру Loxone. Для контроля температуры поверхности пола в стяжке установлены датчики 1-Wire, подключенные к контроллеру Wiren Board.

В доме установлены две приточно-вытяжные установки — по одной на каждый этаж. Вентиляторы трехскоростные, переключение осуществляется через дискретные входы (сухие контакты).

Для управления кондиционерами Daikin используют модули ONOKOM. Они подключены по RS-485 к контроллеру Wiren Board, а уже через него интегрированы в систему Loxone.

В доме используется форсуночное увлажнение с девятью зонами. Каждая зона управляется релейным модулем: при подаче сигнала зона включается, и начинается подача влаги.

### Шторы

В доме используются несколько типов приводов штор:

* с управлением по сухому контакту;  
* с фазным управлением;  
* Wi-Fi-приводы, интегрированные через SmartLife.

Из-за этого логика управления различается. Шторы с управлением по сухому контакту можно только открыть или закрыть — остановка в промежуточном положении не предусмотрена. Приводы с фазным управлением позволяют задавать положение.

Часть штор работает автоматически: контроллер ориентируется на время восхода и заката солнца — утром шторы открываются, вечером закрываются. 

### Контроль доступа

В доме установлены электромеханические замки с беспроводным управлением. Они управляются с помощью небольших брелоков, которые передают команды по Bluetooth. Интегрировать такие замки напрямую в систему автоматизации невозможно — у них нет стандартных интерфейсов. Поэтому брелоки размещены рядом с дверьми и к ним подведено электропитание. Контакты кнопок выведены на реле: при срабатывании реле замыкает контакт и имитирует нажатие кнопки.

### Уличные системы

На улице осуществляется управление обогревом крыльца и водостоков.

В стяжке крыльца и зоне парковки проложен контур с теплоносителем (антифризом), который предотвращает образование наледи. В водостоках установлен электрический нагревательный кабель.

Подогрев включается только при риске образования льда: контроллер анализирует температуру воздуха и наличие осадков. Если температура находится в диапазоне −5…+5 °C и зафиксированы осадки, система включает нагрев.

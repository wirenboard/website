Объект: частная дача с автоматикой на базе Wiren Board. Автор проекта: Александр Хлебников.

| Оборудование | Функции |
| :---- | :---- |
| [Wiren Board 8.5](https://wirenboard.com/ru/catalog/kontrollery/) | Контроллер |
| Плата Pro Micro nRF52840 \+ SX1262 | Meshtastic-узел |

> «Wiren Board хорошо подошел для этой задачи: Linux на борту, MQTT и возможность быстро поднять собственный сервис»
>
> **Александр Хлебников, автор проекта**

Регулярные проблемы с мобильным интернетом и сотовой связью поставили вопрос ребром: либо организовать резервный канал связи, либо оставить дачу без удаленного контроля. Александр решил эту задачу с помощью существующей городской сети Meshtastic — mesh-сети для обмена сообщениями по радиоканалу LoRa. 

Автоматизация дачи построена на контроллере Wiren Board 8.5. Рядом с ним установили Meshtastic-узел на базе платы Pro Micro nRF52840 с LoRa-модулем SX1262. Контроллер взаимодействует с ним через serial/API.

На Wiren Board запустили сервис обработки сообщений. Он принимает команды от Meshtastic-узла, проверяет источник и, если тот входит в список доверенных, сохраняет сообщение в SQLite, разбирает его содержимое и публикует соответствующую команду в MQTT. После этого контроллер отправляет пользователю подтверждение. Дальнейшую обработку выполняют штатные средства Wiren Board — движок правил wb-rules. 

Для управления реализовали набор текстовых команд: включение и выключение света, управление бойлером, включение режима охраны и запрос текущего состояния объекта.

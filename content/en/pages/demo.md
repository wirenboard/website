A public [WB-demo-kit v.3](https://wirenboard.com/ru/product/WB-demo-kit/) is installed in our office. The demo suitcase is connected to Wiren Board Cloud in a special demo account, which you can enter by clicking the "Demo Account" button on the [login page](https://wirenboard.cloud/login). If you already have an account in Wiren Board Cloud, you must first log out.

You can enter the web interface and the controller console to configure the system yourself, write rules or create a dashboard.
